About the Simulator

Although appearing fairly complex, the simulator is designed to follow a fairly straightforward process:
  1. Adding jobs and tests to perform
  2. Running the tests
  3. Viewing the results
Setting up the simulation is done on the Simulation Data Panel. Here you can:
  • Add jobs using the buttons or menus
  • Import jobs from other sources
  • Generate jobs
  • Add tests using the buttons or menus
For more information on using the data panel see the linked article.
