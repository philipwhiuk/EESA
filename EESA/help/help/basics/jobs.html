Jobs

Jobs are perhaps the most important part of the system. They represent an amount of work to be done by a processor and having the following characteristics:
  • Arrival Time
  • Deadline Time
  • Weight
From these basic properties, 2 others can be calculated:
  • Length [Deadline - Arrival]
  • Density [Weight / Length]
Jobs can be added to the system by using the Add Job Window as well as by generating them and importing them from other sources.

Results

When a simulation is run, each test must assign each job to one of the processors. This process of Job Assignment is handled by the selected Job Assignment Algorithm.

The results of this process are then shown on the Results Data Panel, which provides tabulated data and visualisations of the results of the tests performed.
