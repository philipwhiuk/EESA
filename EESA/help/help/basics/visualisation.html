Visualisation

A key component of the system is the ability to view data, both before and after running the algorithms. This is done in two forms:
  • Tabulated Data
  • Charts

Data Tables

The tables in the system are designed to allow the user to manipulate them. Clicking on column headings allows the user to sort them.

The Jobs table also allows field-based filtering using comparison operators.

The Job Results and Time Periods tables allow filtering by Test and Processor.

Charts

There are three types of data output chart which allow viewing of results in different ways. Each chart is also filtered by the same controls as the output charts and can be split into smaller charts via certain options. Charts will automatically resize to the window and data values.

The Job Chart shows the time-line of the allocated job results.

Speed Charts and Energy Charts show the energy usage in each time period.
